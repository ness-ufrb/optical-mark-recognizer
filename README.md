# optical-mark-recognizer

## Installation and configuration

* Versão do python: 3.9

* Criação de venv

Linux/Mac OS:

```
cp env.example .env
python -m venv venv
source venv/bin/activate
pip install -r ./requirements.txt
cp env.example.v2 .env
```

## Save dependencies

```
pip freeze > requirements.txt
```

## Run

```
python omr_v2.py
```
